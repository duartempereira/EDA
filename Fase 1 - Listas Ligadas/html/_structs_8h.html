Detailed Description

Definição das estruturas utilizadas para representar antenas e localizações com efeitos nefastos.

Este ficheiro contém as definições das estruturas de dados para uma antena e uma localização com efeito nefasto. A estrutura Antena é utilizada para armazenar as informações de cada antena, incluindo a sua frequência e coordenadas, enquanto a estrutura Nefasto representa as localizações com efeitos nefastos, com suas respectivas coordenadas.
