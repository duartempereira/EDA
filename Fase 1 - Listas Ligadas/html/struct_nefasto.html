Detailed Description

Representa uma localização com efeitos nefastos no mapa.

Esta estrutura armazena as coordenadas de uma localização que possui um efeito nefasto. A estrutura é utilizada em listas ligadas para armazenar tais posições no mapa.

Parameters
xA coordenada X da localização nefasto.
yA coordenada Y da localização nefasto.
proxPonteiro para a próxima localização nefasto na lista ligada.
